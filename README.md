# Geohash Functions for Elm

A geohash library for Elm.

## Installation

To install it in your development directory:

```sh
elm package install andys8/elm-geohash
```

## Usage

To use it from your code:

```elm
import Geohash

geohash = Geohash.encode -25.38262 -49.26561 8
```
